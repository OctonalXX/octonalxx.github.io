Git For Dummies

1.Choose your provider.

2.Download Github Desktop:

Download Github Desktop Here

3.Run the installer

  1. Run
  2. Skip account making
  3. Set name and email (Required)

4.Cloning process

  1. Provider's Dashboard > Create Repository
  2. Name the repository
  3. Copy clone link
  4. Github Desktop > Clone Repository > URL
  5. Paste clone link
  6. Set local path and press "Clone"

5.Edit files and prepare for first commit

6.Edit summary and press commit

7.Press push origin

8.Use git for username and your password for the password

9.Success! You have made your first commit!

10.Repeat step 5 to 8 for commiting again.
